Harbor 2.1离线部署

发表于 2020-06-12   |   分类于 Harbor   |   阅读次数

环境信息：
系统：CentOS Linux release 7.8.2003
配置：CPU 4核、内存 8G、磁盘200G
docker版本：19.03.13
docker-cmpose版本: 1.27.4
harbor版本：2.1.0

一、部署harbor
1、安装docker

2、安装docker-compose

3、安装harbor
1>数据存放本地方式部署（数据容易丢失，风险较大）

2>外置postgresql、redis、存储使用minio的s3接口、认证使用ldap（pgsql、redis、s3这些可以使用阿里云产品也比较稳定）

#2 部署minio(本次部署仅测试使用，生产环境建议高可用方式部署)

#3 配置harbor.yaml

4、docker-compose维护命令
cd /opt/harbor/
启动harbor: docker-compose up -d
停止harbor: docker-compose stop
重启harbor: docker-compose restart
查看harbor状态: docker-compose ps

5、访问harbor 192.168.86.5 默认用户名密码admin/Harbor12345
